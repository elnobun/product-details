# product-details
A Single Product details page to demonstrate the power of
VueJS 3 in combination with SCSS and CSS Grid, to consume 
an [External API](https://v3-static.supadu.io/dev/products/9780060577315.json)
data to display:
- Title
- Contributor/Author
- Contributor Bio
- Product Image
- Product description
- Prices
- Sale Date
- Reviews
- Retailer Links

## Technology Used
- VueJS 3
- SCSS
- CSS Grid
